Angular Components Overview

  • Main Building Blocks for Angular Applications
    • Components consist of:
      • HTML Template (declares what renders on the page)
      • TypeScript Class (defines behavior)
      • CSS Selector (defines component usage in template)
      • Optionally, CSS Styles (applied to the template)
  • Creating and Configuring Angular Component
    • Live Example / Download Example Link

Prerequisites

  • Install Angular CLI
  • Create Angular Workspace
    • Use "ng new <project-name>"

Creating a Component

  • Best Method: Angular CLI
  • Manual Creation

Creating a Component Using Angular CLI

  • Steps:
    • Navigate to Application Directory
    • Run "ng generate component <component-name>"
    • Default Creation:
      • Component Directory
      • Component File (.component.ts)
      • Template File (.component.html)
      • CSS File (.component.css)
      • Testing Spec File (.component.spec.ts)
    • Additional Info on ng generate component

Creating a Component Manually

  • Steps:
    • Navigate to Project Directory
    • Create <component-name;.component.ts
    • Add Import Statement and Component Decorator
    • Choose CSS Selector
    • Define HTML Template
    • Select Styles for Template
    • Add Class Statement for Component

Specifying a Component's CSS Selector

  • Requirement: CSS Selector
  • Usage Example: <app-hello-world>
  • Adding Selector Property to Component Decorator

Defining a Component's Template

  • Options:
    • External File (templateUrl property)
    • Directly in Component (template property)
    • Multi-line Templates (use backticks)
  • Requirements: Define template using template or templateUrl

Declaring a Component's Styles

  • Options:
    • External File (styleUrls property)
    • Directly in Component (styles property)
  • Styles Property: Array of String CSS Declarations
